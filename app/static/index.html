API Documentation

Base URL

http://your-server:3000/v1

Authentication

Add your API key to the Authorization header:

Authorization: Bearer gm4-your-api-key

Chat Completions

OpenAI-compatible chat endpoint:

curl -X POST http://your-server:3000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer gm4-your-api-key" \
  -d '{
    "model": "gemma-4-31b",
    "messages": [
      {"role": "user", "content": "Xin chào!"}
    ],
    "temperature": 0.7,
    "max_tokens": 2048
  }'

Python (OpenAI SDK)

from openai import OpenAI

client = OpenAI(
    base_url="http://your-server:3000/v1",
    api_key="gm4-your-api-key"
)

response = client.chat.completions.create(
    model="gemma-4-31b",
    messages=[{"role": "user", "content": "Xin chào!"}],
    temperature=0.7,
    max_tokens=2048
)
print(response.choices[0].message.content)

JavaScript (fetch)

const response = await fetch('http://your-server:3000/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer gm4-your-api-key'
  },
  body: JSON.stringify({
    model: 'gemma-4-31b',
    messages: [{ role: 'user', content: 'Xin chào!' }]
  })
});
const data = await response.json();
console.log(data.choices[0].message.content);
